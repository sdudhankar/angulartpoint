Angular Environment Setup



Requirement:

Below are the basic requirement need to install to run any Angular application.

  1. Node JS: It provides runtime environment and lite server to run Angular application. [tsc, webpack used to translate tyopescript]
    Download: https://nodejs.org/en/

    2. Download Node JS Version:Node 6.9.0 or higher.
  2. NPM: NPM is a command line utility. It installs libraries, packages and application which are require to run Angular Application.

  3. Angular CLI: Angular CLI is an command line utility to setup your project easily and fastly.

  4. IDE: Editor for Angular application are: visual studio, webstorm, Atom, Eclipse, Notepad++, subline text.
    Download: https://www.visualstudio.com/downloads/
Visual Studio is a best IDE which comes free of cost from microsoft for Angular Project.
