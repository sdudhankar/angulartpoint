Angular Overview



Angular 2 comes with almost everything you need to build a complicated frontend web or mobile apps, from powerful templates to fast rendering, data management, HTTP services, form handling, and so much more.


Features of Angular are:

  • SPA (single page application)




    Web application or website that fits on a single page and helps to reduce refreshing of page. using SPA, we dynamically rewriting the current page rather than loading entire new pages from a server.
    e.g. GMail, Twitter, Facebook etc.

  • Powerful Data Binding




    Automatic synchronization between Model and the View.

  • Cross platform




    The application which is implemented on multiple computing platforms. It helps tp create desktop-installed apps across Mac, Windows, and Linux. You can build native mobile apps.

  • High Performance/Speed




    The programs will consume less space, taking less rendering time and run quicker.

  • Mobile Oriented




    Web apps built using the mobile toolkit will load on any device, with or without an Internet connection and can take advantage of the searchability, shareability, and no-install-required features of the Web.

  • Gives Our application Clean Structure




    Having separate directory and modules for source code, assets of page(images, 3rd party file,etc.), testing etc.

  • Enhance Productivity




    By supporting different tools like Angular CLI, Templates, IDEs and having consistence in pattern to building the block.

  • Testing




    Having Karma for unit testing, protractor for scenario testing.

  • Universal




    Provides view for PHP, JAVA, node.js, .NET, Python(using Django REST Framework) etc.

  • Animation





  • Modern

    It uses Latest features of Javascript such as classes, module, decorator to build Angular Application.

  • Simplified API

    It has pure built in directives, simple binding etc.
