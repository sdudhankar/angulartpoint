There are two types of Data types
1. Built in
2. User Defined

Built in

Syntax: [access specifier] variable name: Data type;

1. number:

support integer & float value. It also supports hexadecimal, octal, binary number.

2. string:

User Defined

1. Array:

Stored collection of similar data type values.


2. Tuple:

Declare like array but support different types of Data Types.


3. Enum:

To define own data type. an enum is a way of giving more friendly names to sets of numeric values.


Make sure enum is act as a class so don’t declare as a variable inside the class. Enum must declare the outside class.

4. Any:

The Types of variables do not know at compile time and the value may be come from dynamic content that time any data type is a best option.


The any type is a powerful way to work with existing JavaScript.
if we don’t mention data type then by default 'any' is apply to that variable.
