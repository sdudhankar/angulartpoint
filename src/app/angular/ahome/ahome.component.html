What is Angular?

Angular is an open source JavaScript framework to build client-side web applications with the help of HTML, CSS and JavaScript . It is mainly use to create Single Page Application(SPA). It was built by the developers at Google.



Prerequisite:

  1. HTML: describes the structure of Web pages.
  2. CSS: provides visual styles for website.
  3. Java Script: provides interactive functionality for your website.
  4. OOPs Concepts: Helps to write angular applications(class, object, interface etc).

The best example of Angular is www.angulartpoint. website. This site is made itself in Angular.
