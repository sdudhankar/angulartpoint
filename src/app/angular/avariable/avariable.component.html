Variables supports By Angular



Types of variable in Angular can support:


1. Class variable

  • Declare inside the class.
  • It can be private, public, protected.
  • By default is public.
  • this keyword use to access it.


2. Local variable

  • Declare inside the method.
  • Declare by using var or let keyword.
  • By default is public.
  • Scope is limited to block only.


Difference between var and let

Var Let
scop is the nearest function block. scop is to the nearest enclosing block(smaller than a function block).
Visible to the whole function. Visible only in the for loop.
Allow to re-declare the same variable in the same scope. Not Allow to re-declare the same variable in the same scope.


let is only visible in the for() loop and var is visible to the whole function.

3. Static variable

  • Declare inside the class.
  • scope is throughout the class.
