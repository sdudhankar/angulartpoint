Operators



The major operators in Angular 2 Supports are −


1. Arithmetic operators: = , -, *, /, %, ++, --,

2. Logical operators: &&, ||, !

3. Relational operators: >, <, >=, <=, ==, !=

4. Bitwise operators: &, |, ^, ~, <<, >>, >>>

5. Assignment operators: =, +=, -=, *=, /=

6. Ternary/conditional operator: Test? expr1 : expr2
e.g. n>0 ? “pos” : “neg”

7. String operator/concatination: +
e.g. “Hello”+”World” //HelloWorld

8. Type Operator: return data type of operator
e.g. var n=10; console.log(typeof n) //number

9. Negation operator: -
e.g. n=5; p=-n; //-5

10. Instanceof operator: to test if an object is of a specified type or not.
